# 基于大数据的租房与招聘数据分析与可视化系统

## 项目简介
本项目是一款面向互联网行业毕业生及广大求职者的**招聘与租房一体化数据分析与可视化系统**，针对招聘与租房市场信息分散、数据质量良莠不齐、跨平台整合难的问题，融合网络爬虫、数据预处理、机器学习预测、多维度数据可视化技术，实现招聘/租房数据的统一采集、清洗、分析与可视化展示，并提供薪资智能预测功能，为求职者的就业选址、租房决策提供数据支撑。

系统整合拉勾网招聘数据、链家网租房数据，采用**前后端分离架构**开发，前端实现友好的可视化交互界面，后端完成数据处理与业务逻辑，结合 KNN、逻辑回归、朴素贝叶斯算法实现薪资等级预测，打破了传统招聘与租房系统分离的现状，实现信息的一体化整合与分析。

---

## 技术栈

### 后端开发
- 核心框架：Python + Flask
- 数据存储：MySQL（存储用户信息、招聘数据、租房数据）
- 数据库管理：Navicat
- 机器学习：sklearn（实现 KNN、逻辑回归、朴素贝叶斯算法的薪资预测）
- 数据处理：Pandas（数据清洗、预处理、格式转换）
- 网络爬虫：Requests + PyQuery（数据爬取）、fake_useragent（反爬策略）

### 前端开发
- UI 框架：LayUI
- 数据可视化：ECharts
- 开发工具：Visual Studio Code

### 运行环境
- 后端开发工具：PyCharm
- 运行环境：Windows
- Python 版本：3.8+
- Flask 版本：2.0+
- MySQL 版本：5.7+/8.0+

## 系统功能

### 1. 用户信息管理模块
- 账号注册/密码加密登录，支持登录状态校验
- 个人信息编辑（邮箱、个人简介、居住地址、联系电话）
- 密码修改（三步式验证，保障账号安全）

### 2. 招聘数据可视化模块
- **数据概况**：展示全量招聘信息，支持按**学历要求**、**职位名称**精准检索，分页展示详情
- **多维度可视化**：从**薪资、企业、福利、学历、融资、职位类型**6大维度，分**全国/一线城市/新一线城市/二线城市**展示数据
- **热门城市对比**：对北京、上海、广州、深圳、杭州等9大热门城市，实现行政区、学历要求、公司规模、薪资待遇等维度的可视化对比
- **图表类型**：柱状图、折线图、饼图、词云图、矩形树图、漏斗图、南丁格尔玫瑰图等

### 3. 租房数据可视化模块
- **数据概况**：展示全量租房信息，支持按**城市**精准检索，分页展示房源详情（面积、朝向、楼层、价格等）
- **多维度可视化**：从**面积、楼层、朝向、价格**4大维度，分**全国/一线城市/新一线城市/二线城市**展示数据
- **城市房源详情**：对热门城市实现行政区房源分布、面积分布、价格分布等精细化可视化
- **关联分析**：实现房屋**面积与价格**的关联性可视化分析

### 4. 薪资智能预测模块
- 支持用户选择**职位、公司规模、工作经验、学历、城市**等特征
- 提供**KNN、逻辑回归、朴素贝叶斯**三种机器学习模型可选，预测对应岗位的最低薪资区间
- 实时返回预测结果，为求职者的薪资预期提供参考
